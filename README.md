##  Featured Project

### MigraineIQ — Smart Migraine Tracking & ML Insight Web App

MigraineIQ is a calm migraine tracking web application that helps users log migraine episodes, track symptoms and triggers, view ML-based severity insights, connect logs with weather/AQI data, and generate monthly reports.

Key Features:

- User login and signup
- Calm migraine check-in UI
- ML-based migraine severity prediction
- Prediction confidence and explanation
- Weather and AQI integration
- Monthly reports
- CSV export for doctor-friendly tracking

Tech Stack:
Python, Flask, SQLite, SQLAlchemy, Scikit-learn, Pandas, NumPy, HTML, CSS, JavaScript

🔗 Repository: [MigraineIQ](https://github.com/AnamGhafoor/MigraineIQ)
